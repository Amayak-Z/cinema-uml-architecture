# Проектирование архитектуры системы онлайн-кинотеатра

Экзаменационный проект по проектированию архитектуры программного обеспечения. Все схемы выполнены на языке UML с помощью инструмента PlantUML.

## 🗺️ Архитектурные диаграммы

### 1. Сценарии использования (Use Case)
Описывает роли пользователей и ключевые сценарии взаимодействия с системой кинотеатра.
![Use Case](http://plantuml.com)

### 2. Компонентная архитектура (Component)
Показывает внутренние модули системы и связи между ними.
![Component](http://plantuml.com)

### 3. Диаграмма развертывания (Deployment)
Схема физического распределения сервисов и баз данных по серверам.
![Deployment](http://plantuml.com)

### 4. Диаграмма взаимодействия (Communication)
Иллюстрирует динамическое взаимодействие и обмен сообщениями между объектами.
![Communication](http://plantuml.com)

## 🛠️ Технологический стек проектирования
* **Язык моделирования**: UML 2.0
* **Инструментарий**: PlantUML / VS Code
